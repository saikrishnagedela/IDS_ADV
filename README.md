# 🚨 AI-Based Intrusion Detection System (IDS)

## 📌 Overview
This project is an AI-powered Intrusion Detection System that classifies network traffic into multiple attack categories using Machine Learning.

## 🎯 Features
- Multi-class attack detection (DoS, Probe, R2L, U2R)
- Data preprocessing pipeline
- Multiple ML models comparison
- Confusion matrix visualization
- Interactive Streamlit dashboard

## 🧰 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## 📊 Dataset
- NSL-KDD Dataset
